# CareConnect - Healthcare Support Web App

A concept-level web application designed for a Healthcare NGO. It manages volunteer and patient registrations featuring a modern Glassmorphism UI and an integrated AI Chatbot for automated FAQ responses.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Design System:** Glassmorphism, CSS Animations, Responsive Layout
* **Deployment:** Netlify (or Vercel)

## AI & Automation Idea Implemented
1. **AI Chatbot for FAQs:** Integrated a floating, interactive chatbot widget. It uses rule-based natural language matching to instantly answer common queries regarding volunteering, working hours, and contact information, providing a simulated AI assistant experience.
2. **Auto-Data Summary Logging:** Upon form submission, the system automatically captures and logs a structured data summary (Name, Age, Phone, Role) to the console, simulating how data is categorized and routed to respective NGO departments.

## NGO Use-Case
This prototype is built to reduce administrative overhead for healthcare NGOs. Patients seeking help or individuals wanting to volunteer can easily register through a user-friendly interface. Simultaneously, the integrated Chatbot handles repetitive inquiries 24/7, saving valuable human resources and staff time.
